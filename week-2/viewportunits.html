Viewport-relative units: vw, vh, vmin & vmax (and vm)

Deze pagina onderzoekt de werking van viewport-relative units in verschillende browsers.

Viewport-relative units worden ondersteund door IE9 en hoger, Edge, Firefox 19+, Chrome 20+, Safari 6+, iOS Safari 6.1+, Opera 15+ en Android Browser 4.4+. Opera Mini ondersteunt deze units niet.

IE10, IE11 en de Edge browser ondersteunen vmax niet. IE9 doet dat wel, maar ondersteunt vmin niet. In plaats daarvan kun je vm gebruiken. Voor browsers die geen viewport-relative units ondersteunen kan een fallback gebruikt worden voor de CSS attributes waarin viewport-relative units gebruikt worden.

A fallback that would work on most browsers would look like:


.viewport-relative {
	min-width: 30%;
	min-width: 30vmax;
	min-height: 30%;
	min-height: 30vm;
	min-height: 30vmin;
	max-width: 70%;
	max-width: 70vmax;
	max-height: 20em;
	max-height: 70vm;
	max-height: 70vmin;
}

I'm using min-width/height and max-width/height here so that container can adapt to their content within those bounds.
