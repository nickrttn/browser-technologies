Browser features and fallbacks

The pages below describe a range of browser features and provide fallbacks for browsers in which these don't work, or cause problems otherwise. The solutions have been built following the tenets of progressive enhancement and will always try to provide the user with a basic experience first and foremost, and only enhance those if the features required to do so are present in the user's agent of choice.
